# 拼图游戏 v1.0

一个基于Java Swing开发的拼图游戏，具有用户登录注册功能和多种拼图图片选择。

## 功能特点

- **用户管理**：支持用户注册和登录
- **多种拼图图片**：内置多种动物和人物图片供选择
- **交互式界面**：友好的图形用户界面，支持鼠标操作
- **数据持久化**：用户信息保存到本地文件
- **验证码系统**：登录时提供验证码防止机器登录

## 技术栈

- Java SE
- Swing GUI框架
- Hutool工具库（文件操作）
- 文件IO（用户数据持久化）

## 安装与运行

### 环境要求

- Java JDK 8或更高版本
- IDE（如IntelliJ IDEA、Eclipse等）或命令行工具

### 运行步骤

1. 克隆或下载项目到本地
2. 使用IDE打开项目
3. 确保Hutool库已正确添加到项目依赖
4. 运行`LoginJFrame.java`作为入口点
5. 注册新用户或使用已有用户登录
6. 开始游戏！

## 使用说明

### 登录与注册

1. **登录界面**：输入用户名、密码和验证码，点击"登录"按钮进入游戏
   - 验证码可点击更换
   - 点击"注册"按钮进入注册界面

2. **注册界面**：
   - 输入用户名和密码（需重复确认）
   - 点击"注册"按钮完成注册
   - 注册成功后自动返回登录界面

### 游戏操作

1. 登录成功后进入游戏主界面
2. 选择喜欢的拼图图片
3. 使用鼠标点击相邻的拼图块进行交换
4. 完成拼图后会显示成功信息


## 核心功能实现

### 用户管理

- **登录验证**：检查用户名、密码和验证码
- **注册流程**：验证密码一致性，检查用户名唯一性
- **数据持久化**：使用文件存储用户信息，支持程序重启后数据保留

### 拼图游戏

- **图片切割**：将原始图片切割成多个小方块
- **随机打乱**：初始化时随机打乱拼图块顺序
- **拼图验证**：检查拼图是否完成
- **交互操作**：支持鼠标点击交换拼图块

## 开发说明

### 添加新图片

1. 在`image/`目录下创建新的图片分类文件夹
2. 放入完整图片（命名为`all.jpg`）
3. 将图片切割成16个小方块（命名为`1.jpg`到`16.jpg`）
4. 在游戏代码中添加新图片分类的支持

### 修改配置

- 游戏窗口大小、标题等可在各JFrame类的`initFrame()`方法中修改
- 用户数据存储路径可在相关文件操作代码中修改

## 注意事项

1. 请确保图片资源路径正确，否则可能导致界面显示异常
2. 用户数据存储在项目根目录的`userinfo.txt`文件中，请勿手动修改
3. 游戏运行时需要保持图片资源文件夹的完整性
